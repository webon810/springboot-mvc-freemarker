## What you'll need
- JDK 1.8+
- Maven 3+

## Stack
- Java (SE8)
- Spring Boot (v2.0.3)
- FreeMarker

## Local Run
mvn spring-boot:run 

## Local Test (dev)
- SPRING_PROFILES_ACTIVE=dev ./mvnw spring-boot:run
- Visit http://localhost:8585
